# Debit Card Request

## Definition

Use this category when the customer asks for a new, replacement, blocked, unblocked, or reissued debit card.

## Typical Phrases

- new debit card
- replace debit card
- card blocked
- reissue card

## Required Intent

The request must involve debit card issuance, replacement, blocking, or reissue.

## Positive Examples

- Please reissue the debit card for our current account.

## Negative Examples

- Do not use for cheque book requests.

## Similar Categories

- CAT-009 Cheque Book Request

## Confidence Boost Signals

High confidence when debit card appears with issue, replace, block, or reissue.

## Evidence Expectations

Evidence should include debit card request wording.
