# Account Statement Request

## Definition

Use this category when the customer asks for account statements, balance certificate, transaction history, or account ledger copies.

## Typical Phrases

- account statement
- send statement
- transaction history
- balance certificate
- ledger copy

## Required Intent

The request must ask for a statement, certificate, or transaction history.

## Positive Examples

- Please send the account statement for the last six months.

## Negative Examples

- Do not use for payment status inquiries.

## Similar Categories

- CAT-016 Payment Status Inquiry

## Confidence Boost Signals

High confidence when statement, balance certificate, or transaction history appears.

## Evidence Expectations

Evidence should include statement request wording.
