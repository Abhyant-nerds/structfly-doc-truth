# Transaction Limit Increase

## Definition

Use this category when the customer asks to increase payment, transfer, transaction, daily, or beneficiary limit.

## Typical Phrases

- increase transaction limit
- raise transfer limit
- daily limit enhancement
- payment limit increase

## Required Intent

The request must ask for a limit increase or enhancement.

## Positive Examples

- Please increase our RTGS transaction limit.

## Negative Examples

- Do not use for checking whether a payment was completed.

## Similar Categories

- CAT-016 Payment Status Inquiry

## Confidence Boost Signals

High confidence when increase/raise/enhance appears with limit.

## Evidence Expectations

Evidence should include transaction limit wording.
