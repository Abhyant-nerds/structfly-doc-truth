# Cheque Book Request

## Definition

Use this category when the customer asks for a new cheque book, cheque leaves, or replacement cheque book.

## Typical Phrases

- issue cheque book
- request cheque leaves
- new checkbook
- cheque book exhausted

## Required Intent

The request must ask for cheque book issuance or replacement.

## Positive Examples

- Please issue a new cheque book for our current account.

## Negative Examples

- Do not use for debit card requests.

## Similar Categories

- CAT-010 Debit Card Request

## Confidence Boost Signals

High confidence when issue/request language appears with cheque book or cheque leaves.

## Evidence Expectations

Evidence should include cheque book request wording.
